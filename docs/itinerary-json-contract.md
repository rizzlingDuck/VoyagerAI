# Itinerary JSON Contract

Saved trips store the full itinerary response in `public.trips.itinerary` as JSONB. The backend validates generated itineraries with `backend/ai/schemas.js` before a trip can be returned to the UI.

## Table Shape

`public.trips`

| Column | Type | Notes |
| --- | --- | --- |
| `id` | `uuid` | Primary key, generated by Postgres. |
| `user_id` | `uuid` | References `auth.users(id)`, protected by RLS. |
| `destination` | `text` | Display/search value copied from `itinerary.destination`. |
| `itinerary` | `jsonb` | Validated itinerary object. |
| `saved_at` | `timestamptz` | Creation timestamp used for sorting. |
| `updated_at` | `timestamptz` | Maintained by trigger. |
